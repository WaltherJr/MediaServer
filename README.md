# MediaServer
A media server that plays [YouTube](https://www.youtube.com) videos via the `/video/*` API, and [Allente](https://www.allente.se) TV channels via the `/tv/*`API.

### Languages
* [Java](https://en.wikipedia.org/wiki/Java)
* [JavaScript](https://en.wikipedia.org/wiki/JavaScript)
* [C/C++](https://en.wikipedia.org/wiki/C%2B%2B) (via [JNI](https://en.wikipedia.org/wiki/Java_Native_Interface) for native [Win32 API](https://en.wikipedia.org/wiki/Windows_API) access)
* [HTML](https://en.wikipedia.org/wiki/HTML)/[CSS](https://en.wikipedia.org/wiki/CSS)

#### Frameworks
* [Spring Boot](https://spring.io/projects/spring-boot)
* [Thymeleaf](https://www.thymeleaf.org/)

#### Tools and libraries
* [Chrome DevTools Protocol](https://chromedevtools.github.io/devtools-protocol/) - For browser control/embedding in application
* [YouTube Player SDK](https://developers.google.com/youtube/player_parameters)
* [Maven](https://maven.apache.org/)

### API documentation
Can be found at [/v3/api-docs](http://localhost:8000/v3/api-docs) and
[/swagger-ui/index.html](http://localhost:8000/swagger-ui/index.html).
